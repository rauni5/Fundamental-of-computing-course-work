# Retail Management System (Python Coursework)

## Overview

This project was developed as part of the Fundamentals of Computing module. It is a Python-based retail management system that manages product records, inventory information, and bill generation.

## Features

* Product catalogue management
* Inventory tracking
* Customer billing
* Automated bill generation
* File handling
* Data persistence using text files

## Technologies Used

* Python
* Text File Storage

## Modules

* Main.py
* Operations.py
* Read.py
* Write.py
* Display.py
* Generate_bill.py
